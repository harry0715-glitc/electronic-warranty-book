# 楓根室內裝修設計有限公司｜電子保固書

這個專案目前已從原本的「電子名片」範本脫離，並建立成 **電子保固書初始版**。

## 目前提供

- 接近紙本「保固切結書」的正式文件式首頁
- 可直接開啟的電子保固書展示頁
- 案件欄位表格 / 切結聲明 / 廠商資訊 / 用印區骨架
- 預留後續串接報修表單、附件索引、LIFF、Apps Script、GitHub Pages 或其他後台資料來源

## 專案結構

- `index.html`：電子保固書主頁
- `assets/logo-transparent.png`：透明版 Logo
- `assets/logo-header.jpg` / `assets/logo.jpg`：品牌圖像素材
- `assets/fonts/...`：字型資源
- `line-flex-message.json`：保固書分享訊息範例
- `line-push-sample.json`：LINE 推播訊息範例
- `warranty-data.sample.json`：保固切結書資料欄位樣板

## 目前資料狀態

頁面內資料仍為 **示意內容**，例如：
- 案件名稱
- 工程地點
- 承攬範圍
- 結算金額
- 保固日期與切結聲明
- 客服電話 / 報修表單

正式上線前，建議先替換成單一案件實際資料。
